Run the following bash commands to gather context, then greet the user:

1. Run `date` to get the current date and time.
2. Run `pwd` to get the current working directory.

Then display a greeting message to the user in this format:

```
Hello! Welcome to Claude Code.

Date & Time : <output of date>
Directory   : <output of pwd>
```

After showing the greeting, ask the user:

"What would you like to do today?"

Wait for the user's response and help them accordingly.
